# Логотипті орналастыру

Логотипті сайтқа орналастыру үшін:

1. Суретті `public/` папкасына `logo.png` ретінде сақтаңыз
2. Суреттің форматы: PNG (немесе JPG, SVG)
3. Ұсынылатын өлшем: 100x100px немесе одан үлкен (жоғары сапа үшін)

Логотип мына жерлерде пайдаланылады:
- Header (жоғарғы баған)
- Браузер вкладкасындағы иконка (favicon)

Егер суретті сақтағаннан кейін сайтты қайта жүктесеңіз, логотип көрінеді.
